2510560001 - 2510560300

- used，boolean，默认 false
- usedByUserId，可为空，关联用户 id
- usedAt，可为空
- createdAt

1. 新增忘记密码入口

用户忘记密码后，可以通过以下信息重置密码：
- 邀请码
- 用户昵称
- 新密码
- 确认新密码

逻辑要求：

- 用户昵称必须与对应账号匹配。

- 不要提供“查询密码”功能，只允许重置密码。

- 如果项目使用 Better Auth，请优先调用或复用 Better Auth 推荐的密码更新/哈希逻辑。
